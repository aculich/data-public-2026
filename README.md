# data-public-2026

> Public data repository for the 2026 vintage toolchain constellation

## Purpose

This repository contains generic, non-identifying data that can be safely shared publicly. This includes:

- Research artifacts (HTML, markdown notes)
- Generic documentation
- Non-identifying examples and templates

## Structure

```
data-public-2026/
├── research/          # Research artifacts from ideas/
├── templates/         # Public templates
└── examples/          # Generic examples
```

## Privacy Policy

All data in this repository is:
- ✅ Non-identifying
- ✅ No PII (Personally Identifiable Information)
- ✅ No system-specific paths
- ✅ No serial numbers or device identifiers
- ✅ Scanned with gitleaks before commit
